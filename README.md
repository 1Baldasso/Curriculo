# Currículo React
Esse projeto foi um estudo de HTML, CSS e React que me deixou muito feliz com a flexibilidade e a facilidade em gerar novas versões rapidamente.

## Objetivo
Ao tentar atualizar o meu currículo me deparei com um problema, eu não tinha mais Word e nem o template que utilizei pra fazer o meu primeiro currículo, dessa forma, decidi criar meu próprio currículo em React para treinar e ao mesmo tempo ter um controle maior sobre os componentes do currículo.

## Como utilizar o Template
- Copie a pasta do projeto para seu computador e modifique da forma que desejar. 

- Os dados da pessoa (onde você vai trocar meus dados pelos seus) estar em


```./src/Assets/PersonInfo.js``` para informações em inglês


```./src/Assets/PersonInfoPTBR.js``` para informações em português

- Troque também a imagem substituindo o arquivo ProfilePic.jpg

- Dados como link para portfolio e skills devem ser trocados diretamente no código

- Após esses passos, no seu terminal, na pasta do projeto, execute o comando

- Para definir a linguagem utilizada no currículo que você vai gerar, a propriedade ```language``` deve ser mudada no arquivo ```App.js```

```
npm install
```

- Após isso você esta pronto para executar o projeto com o comando

```
npm start
```

- Ao abrir a página simplesmente clique com o botão direito e em "Imprimir" ou simplesmente utilize o atalho Ctrl+P

- Salve o arquivo como PDF (Utilize a opção salvar como PDF, opções Microsoft Print to PDF e outras salvam sem os hiperlinks desejados)


### Disclaimer:
This repository is a HTML based curriculum which works when the page is printed or a PDF is created with the page view, it is not meant to be a functional website
